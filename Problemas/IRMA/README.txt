Dados do problema IRMA.

Arquivos:
- ExpressionData_IRMA_1.csv: arquivo de dados de expressão gênica. Cada linha é iniciada pelo nome do gene (espécie) e os demais valores são as concentrações. As colunas são as células/tempo de simulação.
- PseudoTime.csv: arquivo de pseudotime. A primeira coluna é o nome da célula e a segunda coluna é o valor de tempo.
- refNetwork.csv: rede ground-truth. Primeira coluna é o regulador, segunda coluna é o alvo e a terceira coluna é o tipo de regulação (+ = ativação / - = inibição).


Detalhes sobre esse modelo e os papers de referência estão apresentados na tese.
